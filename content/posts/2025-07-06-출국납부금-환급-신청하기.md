2024년 7월부터 **출국납부금**이 기존 10,000원에서 **7,000원으로 인하**되었고, 어린이(12세 미만)는 **전액 면제**되었습니다. 그러나 많은 분들이 공항에서 자동 환급되지 않아 **출국납부금을 환급받지 못하는 사례**가 계속되었죠. 이제는 **국제항공료 인하분을 돌려받기 위한 절차**가 간소화되어 환급을 받을 수 있습니다. 그 중 핵심은 **정확한 항공편명을 포함한 여행 정보**를 확보하는 것입니다.

## 출국납부금 환급 신청하기 {#출국납부금-환급-신청하기.wp-block-heading}

### ❗ 그런데, 항공편명은 어디에? {#❗-그런데-항공편명은-어디에.wp-block-heading}

**출입국 사실 증명서**를 발급받으면 출국 날짜 및 방법은 알 수 있지만, **항공편명과 국가명은 기재되지 않습니다.** 따라서 환급 신청 시 반드시 항공편명(OZ123, KE901 등)을 별도로 확인해야 합니다.

### 이메일로 항공편명 찾기 {#이메일로-항공편명-찾기.wp-block-heading}

이처럼 **정확한 출국 날짜+이메일 하나면 충분**합니다. 예약번호 대신 **영문 이름, 출국일, 편명, 항공사 정보**만 있으면 환급 가능하기 때문이죠.

<hr class="wp-block-separator has-alpha-channel-opacity" />

### 🧾 출국납부금 환급 신청 총정리 {#🧾-출국납부금-환급-신청-총정리.wp-block-heading}

#### ① **환급 대상 요건** {#①-환급-대상-요건.wp-block-heading}

<ul class="wp-block-list">
  <li>
    <strong>항공권 발매일</strong>: 2024년 6월 30일 이전
  </li>
  <li>
    <strong>출국일</strong>: 2024년 7월 1일 이후
  </li>
  <li>
    <strong>성인</strong>: 최대 3,000원 환급
  </li>
  <li>
    <strong>만 12세 미만 어린이</strong>: 10,000원 전액 환급 (2025년 8월 1일부터 신청 가능)
  </li>
</ul>

#### ② **환급 가능한 금액** {#②-환급-가능한-금액.wp-block-heading}

<ul class="wp-block-list">
  <li>
    <strong>성인</strong>: 3,000원
  </li>
  <li>
    <strong>어린이</strong>: 10,000원
  </li>
</ul>

#### ③ **신청 사이트** {#③-신청-사이트.wp-block-heading}

<ul class="wp-block-list">
  <li>
    <a href="https://tour-refund.kr/">tour‑refund.kr</a> (한국공항공사 공식 프로그램)
  </li>
</ul>

#### ④ **필수 입력 정보** {#④-필수-입력-정보.wp-block-heading}

<ul class="wp-block-list">
  <li>
    영문 이름(여권 기준), <strong>출국일</strong>, <strong>항공사명</strong>, <strong>항공편명</strong>, <strong>예약번호 불필요</strong>, <strong>환급받을 계좌</strong>
  </li>
</ul>

#### ⑤ **환급 절차** {#⑤-환급-절차.wp-block-heading}

<ol class="wp-block-list">
  <li>
    사이트 접속 → 본인인증
  </li>
  <li>
    정보 입력 (여권영문명‧항공사‧편명‧출국일‧계좌)
  </li>
  <li>
    신청 완료 → 문자 통지
  </li>
  <li>
    1~3일 내 환급 완료
  </li>
</ol>

#### ⑥ **주의사항** {#⑥-주의사항.wp-block-heading}

<ul class="wp-block-list">
  <li>
    미성년자(12세 미만)는 2025년 8월 1일부터 신청 가능
  </li>
  <li>
    <strong>사칭 사이트 주의</strong>: 반드시 공식 tour-refund.kr 이용
  </li>
  <li>
    <strong>환급 계좌명의</strong>는 여권 영문명과 <strong>정확히 일치</strong>해야 함
  </li>
  <li>
    <strong>가족 환급</strong>은 각자 신청해야 함
  </li>
</ul>

### 🧭 이메일로 여행 정보 확인하는 꿀팁 {#🧭-이메일로-여행-정보-확인하는-꿀팁.wp-block-heading}

<ol class="wp-block-list">
  <li>
    <strong>메일함 검색 키워드</strong>: 항공사명, &#8220;e-ticket&#8221;, &#8220;boarding pass&#8221;
  </li>
  <li>
    <strong>항공권 PDF 첨부 파일</strong> 클릭 → 날짜, 편명, 출발지/도착지 확인
  </li>
  <li>
    <strong>예약번호</strong>가 없어도 상관없음 — 항공편명(OZ123)만 입력하면 됩니다
  </li>
  <li>
    <strong>이메일·캘린더 내역이 없다면</strong>: 항공사 마일리지 또는 앱에 로그인 후 <strong>탑승 내역 조회</strong> 시 편명 확인 가능
  </li>
</ol>

## ✅ 마무리하며 {#✅-마무리하며.wp-block-heading}

<ol class="wp-block-list">
  <li>
    <strong>출국납부금은 자동 환급되지 않습니다</strong>. 반드시 신청해야 돌려받을 수 있습니다.
  </li>
  <li>
    <strong>출입국 사실 증명서는 날짜만</strong>, 편명은 <strong>e-ticket 이메일이나 앱에서 확인하세요.</strong>
  </li>
  <li>
    <strong>필수 정보만 정확히 입력하면, 5분 내로 신청 가능</strong>하고 1~3일 내 계좌로 입금됩니다.
  </li>
  <li>
    <strong>성인 3,000원, 어린이 10,000원</strong>은 절대 작은 금액이 아닙니다.
  </li>
  <li>
    <strong>미성년자 환급도 가능해진 2025년 최신 기준</strong>을 놓치지 마시고, <strong>공식 사이트에서 안전하게 신청하세요.</strong>
  </li>
</ol>
